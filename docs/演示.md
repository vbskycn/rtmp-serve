

## 新做的拉流推流系统



```
# 拉取镜像
docker pull zhoujie218/rtmp-serve:latest

# 运行容器(持久化存储)
docker run -d \
  --name rtmp-serve \
  --restart unless-stopped \
  -p 3000:3000 \
  -v $(pwd)/config:/app/config \
  -v $(pwd)/streams:/app/streams \
  -v $(pwd)/logs:/app/logs \
  -e NODE_ENV=production \
  -e TZ=Asia/Shanghai \
  zhoujie218/rtmp-serve:latest

# 运行容器(无持久化)
docker run -d \
  --name rtmp-serve \
  --restart unless-stopped \
  -p 3000:3000 \
  -e NODE_ENV=production \
  -e TZ=Asia/Shanghai \
  zhoujie218/rtmp-serve:latest
  
```
